# ObjectPool

A generic, strongly-typed object pooling library for Roblox Luau.

## English

### Overview
`ObjectPool` optimizes performance by reusing heavy instances.

### Installation
```toml
[dependencies]
ObjectPool = "cometahn142/objectpool@^0.1"
```

### 개요
